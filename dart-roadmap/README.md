# Dart Learning Path (Beginner → Advanced)

This repository is a practical, project-based path to learn Dart from zero to advanced. Work through the modules in order. Each module has a short guide and exercises you can run locally or paste into DartPad.

- Recommended pace: 5–8 hours/week over 4–6 weeks
- Practice-first: each topic includes small, focused exercises
- No Flutter required (pure Dart). Great foundation if you later learn Flutter

## How to use
1) Read the module `README.md`.
2) Complete the exercises in `exercises/*.dart`.
3) Run locally or in DartPad.

## Install Dart (Linux)
See `00_setup/README.md` for options. Quick verify: `dart --version`.

## Roadmap overview
- 00_setup: Install, editor, run, format, analyze
- 01_basics: Syntax, types, null-safety, functions, collections
- 02_oop: Classes, constructors, abstract, mixins, extensions, enums
- 03_generics_collections: Generics, iterables, records, collection methods
- 04_async: Futures, async/await, Streams, error handling
- 05_isolates: Concurrency with isolates and message passing
- 06_tooling_testing: Pub, lints, format, analyze, testing
- 07_patterns_records: Pattern matching, records, sealed classes
- 08_performance: Build, compile, profiling, benchmarks
- 10_projects: Capstone project ideas

## Weekly plan (suggested)
- Week 1: 00_setup, 01_basics
- Week 2: 02_oop
- Week 3: 03_generics_collections
- Week 4: 04_async
- Week 5: 05_isolates, 06_tooling_testing
- Week 6: 07_patterns_records, 08_performance, start a project (10_projects)

## Run code
- Single file: `dart run path/to/file.dart` or `dart path/to/file.dart`
- Console project: `dart create -t console my_app && cd my_app && dart run`
