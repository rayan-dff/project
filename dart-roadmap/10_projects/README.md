# 10_projects — Capstone ideas

Pick one and build it end-to-end. Focus on clean code, tests, and performance.

## Ideas
- CLI Todo manager with persistence (JSON file)
- URL shortener (in-memory map + basic HTTP using `shelf`)
- Text analyzer (word frequency, n-grams, streaming large files)
- Markdown to HTML converter (parsing + formatting)

## Tips
- Start with a console app: `dart create -t console_full app`
- Add tests early. Keep functions small and pure.
- Lint and format: `dart analyze` and `dart format .`
