# 01_basics — Language fundamentals

Learn core syntax and semantics. Complete exercises in `exercises/`.

## Topics
- Variables, `var`, explicit types, `final`, `const`
- Null-safety: `?`, `!`, `??`, `??=`, null-aware access `?.`
- Numbers, strings, booleans
- String interpolation, multi-line strings
- Control flow: `if/else`, `switch`, `for`, `while`, `break`, `continue`
- Functions: params (positional, named, required), arrow functions, default values
- Scope, lexical closures
- Collections: `List`, `Set`, `Map`, spread `...`, control-flow in collections

## Read
- Language tour: https://dart.dev/language
- Null safety: https://dart.dev/null-safety

## Do
- Open `exercises/` and solve in order. Run each with `dart run <file>.dart`.
